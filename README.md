# YatraDristi – Intelligent Road Safety & Risk Prediction System

YatraDristi is a hybrid ML + rule-based road safety and accident risk prediction system built for B.Sc. CSIT 7th semester project work.

## Tech Stack

- Python
- Streamlit
- Pandas
- NumPy
- Scikit-learn
- Folium
- OpenWeather API
- OpenStreetMap API
- SQLite (optional)

## Core Features

- Route-based risk prediction
- Segment-wise risk analysis
- Random Forest based accident risk classification
- Real-time weather-based risk adjustment
- Route risk aggregation
- Accident hotspot visualization
- Safety alerts
- Interactive map dashboard
